Methodological Notes

Annotation Pipeline

A Python script (annotate_uzb_snps.py) performed the following steps for each of the 490 Uzbek-specific SNPs:

  1. Ensembl VEP (GRCh38) — batch POST to /vep/human/region (100 per request): returns rsID, gene symbol, consequence terms, SIFT, PolyPhen scores.
  2. GWAS Catalog REST API — /singleNucleotidePolymorphisms/{rsid}/associations: returns phenotype/disease associations with p-values and PubMed IDs.
  3. NCBI ClinVar (eutils) — esearch + esummary: returns clinical significance (benign / pathogenic / VUS) and associated conditions.
  4. GTEx v8 API — /association/singleTissueEqtl queried for 5 tissues (Whole Blood, Liver, Artery Aorta, Frontal Cortex, Subcutaneous Adipose): eQTL p-values and effect sizes (NES).

Rate limiting: ≤15 req/s (Ensembl), ≤3 req/s (NCBI), standard for GWAS/GTEx. Exponential back-off on HTTP 429. Results checkpointed to JSON after each phase.

Key Finding: Most PBS Candidates Lack dbSNP/GWAS Entries

Ensembl VEP returned rsid='.' (not found in dbSNP) for the large majority of the 490 high-PBS variants. This is scientifically expected and informative: variants with extreme PBS scores in Uzbek — where the ancestral allele is retained at low frequency while it approaches fixation globally — are by definition ultra-rare worldwide and are unlikely to appear in reference variant catalogues built primarily from European and East Asian populations. Consequently, GWAS Catalog and ClinVar queries returned no hits for most variants, consistent with the conclusion that these are genuinely novel or under-catalogued population-specific variants.

The exceptions — variants with known rsIDs (e.g., rs2095906645 in SPI1, rs2127467025 in MOG, rs923936635) — are the highest-priority candidates for followup, as they are detectable by standard genotyping arrays and have at least partial representation in global databases. GTEx coordinate-based lookups (not rsID-dependent) were unaffected and provide eQTL data where signals exist.

Interpretation Caveats

SNP coordinates are GRCh38. Near-private calls are MAF_UZB ≥ 5% with all reference populations ≤ 1%. SIFT/PolyPhen scores are algorithm predictions and require experimental validation. GTEx eQTL significance is nominal (p < 0.05); tissue-specific expression context should be consulted. GWAS and ClinVar lookups depend on rsID availability; variants absent from dbSNP will show no hits regardless of functional importance.
