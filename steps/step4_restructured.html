Step 4: Michigan Imputation Server

Cloud-based genotype imputation using 1000G reference panel

✓ Completed — December 18-22, 2025

1. Overview

Michigan Imputation Server is a cloud-based service that performs state-of-the-art genotype imputation using phasing (EAGLE2) and imputation (Minimac4) algorithms with the 1000 Genomes Project reference panel.

Impact: Increases variant density from 473K typed variants to 10.8M variants, enabling detection of rare variants and improving statistical power for downstream analyses.

4. Commands Executed

Step 1: Upload to Michigan Server

for i in {1..22}; do upload_to_server chr${i}.vcf.gz; done

Step 2: Server Processing Parameters

  • Reference Panel: 1000G Phase 3 v5 (hg38)
  • Phasing: EAGLE2
  • Imputation: Minimac4
  • Mode: Standard (with quality control)

Step 3: Download Results

for i in {1..22}; do download_from_server chr${i}.dose.vcf.gz; done
