1. Overview

This page defines the phenotype classification for the RPL (Recurrent Pregnancy Loss) GWAS. Each of the 1,815 genotyped subjects has up to 20 pregnancy slots with outcome codes, gestational ages, and other metadata. Classification is derived entirely from pregnancy outcome data — original spreadsheet labels are not used.

8. Proposed Case/Control Criteria

CASE (RPL)

  • clean_loss_count ≥ 2
  • Loss = outcome ∈ {2, 3, 6} AND GA < 20 weeks
  • Age 18–45 at enrollment
  • No code 8 in current slot OR threshold already met from prior pregnancies
Result: 398 definite + 27 probable = 425 cases

CONTROL

  • clean_loss_count = 0
  • Live births (code 1) ≥ 1
  • Age 18–45 at enrollment
  • No code 8 pregnancy or confirmed delivered since
Result: 861 definite + 6 probable = 867 controls

GRAY ZONE

Exactly 1 intrauterine loss. Exclude from both case and control in primary analysis. Optional: secondary "sporadic loss" sub-analysis.
