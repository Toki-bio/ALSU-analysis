# Practical Strategy for the 69-Sample Rescue Panel
Date: 2026-04-29
## Plain-English Strategy
A 96-SNP identity panel is not a replacement GWAS chip. It is a small identity barcode. The lab genotypes the same 96 highly informative SNPs in each original DNA tube. We then compare that 96-SNP barcode to the existing GSA genotype data. If the barcode matches the current GSA sample, the label is verified. If it matches a different GSA sample, the label is wrong. If it matches no current genotype, that original tube may represent a missing individual and can be full-regenotyped.
The panel should be built from SNPs already present on the Illumina GSA data. That gives direct apples-to-apples matching against the existing PLINK genotypes. A generic fingerprint panel is only acceptable if the vendor confirms substantial overlap with the GSA SNPs or can genotype the exact custom SNP list.
## Where To Order It
Ask for a custom SNP genotyping or identity fingerprinting service, not necessarily a literal chip. Practical technologies are LGC Biosearch KASP, Agena MassARRAY iPLEX, Thermo Fisher TaqMan/OpenArray, or a local university/genomics core that offers custom SNP genotyping. Give them the rsID, hg19 coordinate, alleles, and ideally flanking sequence if they request assay design input. For 69 samples x 96 SNPs, a full Illumina array is usually overkill.
## Draft 96-SNP Panel
I generated a draft list from the cleaned GSA data: `data/rescue_96snp_panel_candidates.tsv`. Selection criteria: autosomal SNPs already in `ConvSK_mind20_dedup_snpqc`, LD-pruned with PLINK `--indep-pairwise 200kb 50 0.2`, rsID present, A/C/G/T non-palindromic alleles, MAF >= 0.40, F_MISS <= 0.002, and spread at least 5 Mb apart where possible. Vendor assayability still needs confirmation; replace any failed assay with the next SNP selected by the same rules.

## Matching Rule After Results Return
For each tube fingerprint, compare against all 1,247 existing GSA genotypes using only the 96 panel SNPs. Count concordant non-missing genotypes. A clean match should have near-complete concordance; one or two discordant calls can be tolerated depending on assay quality, but multiple mismatches should trigger manual review. Do not compare only to the nominated partner, because a swap can resolve to a third sample.
